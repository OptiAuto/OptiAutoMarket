Mesures implémentées
1. Authentification JWT (jsonwebtoken)
Le mot de passe n'est plus jamais renvoyé au client
Un JWT signé HS256 avec expiration de 4h est généré
Le JWT contient un jti unique par session
Stockage dans sessionStorage (détruit à la fermeture du navigateur) au lieu de localStorage
2. Hachage bcrypt (12 rounds)
Le mot de passe admin est haché au démarrage du serveur
Comparaison via bcrypt.compareSync (timing-safe)
3. Helmet (headers de sécurité)
Content-Security-Policy : restreint les sources de scripts, styles, images, connexions
Strict-Transport-Security (HSTS) : force HTTPS pendant 1 an
X-Frame-Options: DENY : bloque l'inclusion en iframe
X-Content-Type-Options: nosniff : empêche le MIME sniffing
Referrer-Policy: strict-origin-when-cross-origin
Permissions-Policy : désactive caméra, micro, géoloc, paiement
4. Rate Limiting (express-rate-limit)
Login : max 5 tentatives / 15 minutes par IP
API : max 60 requêtes / minute
5. Protection CSRF
Token CSRF généré à chaque connexion
Vérifié sur toutes les requêtes mutantes (POST, PUT, PATCH, DELETE)
Comparaison timing-safe via crypto.timingSafeEqual
6. Protection XSS (côté client)
Zéro innerHTML avec des données utilisateur
Tout est construit via document.createElement + textContent
Plus aucun onclick inline dans le HTML
Fonction d'échappement esc() disponible
7. Validation / Sanitisation serveur
sanitizeStr() : supprime <>, tronque à 1000 chars
sanitizeHtml() : supprime <script>, onerror=, javascript:
validateId() : vérifie que les IDs sont des entiers positifs
validateImageBuffer() : vérifie les magic bytes (JPEG/PNG/WebP/GIF)
Noms de fichiers nettoyés (caractères alphanumériques uniquement)
Liste blanche des clés de métafields autorisées
8. Audit Logging
Chaque action (login, création, modification, suppression) est loggée avec timestamp et IP
9. Autres
Serveur écoute uniquement sur 127.0.0.1 (pas accessible depuis le réseau)
X-Powered-By désactivé
Cache désactivé (no-store)
robots: noindex, nofollow sur l'admin
HPP (HTTP Parameter Pollution) protection
Limite de taille des requêtes (1 Mo JSON, 8 Mo images)
.gitignore renforcé (.env, node_modules/, clés, logs)
Aucun token Shopify exposé dans le code front (vérifié par grep)
