# Configuration des filtres véhicules

## Tags à utiliser sur les produits

Pour que les filtres fonctionnent correctement, chaque véhicule (produit) doit avoir les **tags** suivants dans Shopify :

| Filtre | Format du tag | Exemple |
|--------|---------------|---------|
| Marque | `marque:NomMarque` | `marque:Renault` |
| Modèle | `modele:NomModele` | `modele:Clio` |
| Carrosserie | `carrosserie:Type` | `carrosserie:Citadine` |
| Carburant | `carburant:Type` | `carburant:Diesel` |
| Boîte | `boite:Type` | `boite:Manuelle` |
| Année | `annee:AAAA` | `annee:2022` |
| Kilométrage | `km:XXXXX` | `km:45000` |

## Activer le filtrage dans Shopify

1. Aller dans **Paramètres** > **Thèmes** > **Personnaliser**
2. Ouvrir une page **Collection**
3. Dans les paramètres de la collection, activer **Filtrage par tags**
4. S'assurer que la collection "Tous les produits" ou la collection cible a le filtrage activé
